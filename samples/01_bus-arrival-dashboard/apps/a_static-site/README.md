# a_static-site（静的サイト版）

博多→九大 通学時間ダッシュボードの**静的サイト実装**です。`index.html` 1ファイルで完結しており、HTML + CSS + JavaScript（Chart.js はCDN経由）で動きます。ビルド不要、サーバー不要で、ブラウザで開くだけで動作します。

## このファイルの中身

- **タイムピッカー型シミュレーター**: 到着希望時刻を選ぶと、推奨出発時刻と所要時間を表示
- **時間帯別所要時間グラフ**: Chart.js で朝ラッシュ〜夜間の所要時間の推移を折れ線表示
- **区間別テーブル**: 博多→糸島の区間ごとに距離と所要時間を表示

データは `index.html` 内にダミーの静的データとしてハードコードされています（国交省CSVから事前集計した想定の値）。

## 起動方法

### 方法1: ファイルを直接開く

`index.html` をローカルPCにダウンロードして開いてもOKです。

### 方法2: Python の簡易サーバで起動

ローカルサーバ経由で動かしたい場合は、このディレクトリで以下を実行します。

```bash
python3 -m http.server 8000
```

ローカルPCのブラウザで http://<あなたのcode-serverのIPアドレス>:8000 を開くと表示されます。code-server のターミナルで `Ctrl + C` とすることで停止できます。
